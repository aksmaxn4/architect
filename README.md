# Вариант №...  
Информационная система для мониторинга арбитражных ситуаций между централизованными (CEX) и децентрализованными (DEX) криптовалютными биржами

Трейдеры стремятся иметь единый инструмент для автоматизированного отслеживания ценовых расхождений между разными криптовалютными биржами, чтобы оперативно выявлять арбитражные возможности. Из-за высокой волатильности рынка и частого обновления котировок ручной мониторинг становится малоэффективным. Система должна собирать данные с нескольких торговых площадок, сравнивать их и предоставлять пользователю релевантные торговые сигналы.

---

## Пользователи
- **Трейдеры** — сотни активных пользователей, регулярно ищущих мгновенные ценовые расхождения.  
- **Администратор/аналитик платформы** — небольшая команда, отвечающая за поддержание актуального списка бирж, API-ключей и корректность данных.

---

## Требования
- система получает рыночные данные (цены, объемы, ликвидность) с поддерживаемых централизованных и децентрализованных бирж в режиме, близком к реальному времени;  
- система автоматически сравнивает котировки между биржами и определяет ценовые расхождения, превышающие заданные пользователем пороги;  
- пользователи могут настраивать параметры фильтрации (минимальный спред, объем сделки, ликвидность, торговые комиссии);  
- система отправляет уведомления о появлении арбитражных возможностей (интерфейс, e-mail, мессенджеры);  
- пользователи могут просматривать историю обнаруженных арбитражных ситуаций, включая исходные биржи, относительный спред, ликвидность и комиссионные расходы;  
- пользователи могут экспортировать результаты мониторинга в форматах CSV/JSON или получать доступ через API для интеграции с торговыми ботами;  
- администраторы могут добавлять и удалять биржи, изменять частоту обновления данных, управлять API-ключами и конфигурациями мониторинга;  
- интерфейс системы должен визуализировать текущие расхождения в виде интерактивного дашборда (таблицы, графики, индикаторы спреда и ликвидности);  
- система должна учитывать сетевые комиссии (например, gas fee на DEX), влияющие на конечную арбитражную выгоду.

---

## Дополнительный контекст
- рынок криптовалют высоковолатилен: арбитражные возможности существуют очень короткое время — доли секунды;  
- конкуренция среди арбитражных трейдеров и автоматизированных ботов интенсивная, поэтому задержки в обработке данных критичны;  
- различные биржи имеют свои ограничения API (скорость запросов, форматы данных, лимиты), что усложняет работу системы;  
- система должна быть частью более широкой инфраструктуры автоматизированного трейдинга;  
- точность, своевременность и надежность данных напрямую влияют на потенциальную прибыль пользователей.
